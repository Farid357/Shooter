# **О проекте**
  <font size="48"> Переживайте волны и побеждайте врагов! На вашем пути встретятся, как обычные скелеты, так и големы, огромные пауки и многие другие! 
  Сможете ли победить их всех под убойную музыку?

### Подходы
 - В проекте используется архитектурный подход **MV**, в котором модель знает о view
 - Проект пытается следовать GRASP & SOLID
 - Чистый DI (без Zenject, VContainer and etc)
 - Composite Root
 
### Были разработанны:
 - Стрельба
 - Разнообразыне враги
 - Инвентарь
 - Меню настроек с выбором языка
 - Магазин, в котором можно покупать новые типы оружий
 - Разные валюты
 - И многое другое!

### Плагины
  - Odin Inspector
  - UniTask
  - DoTween
  
# Итог
Этот проект приблизил меня к пониманию **ООП**, в ходе его разработки было реализованно множество механик, что дало ощутимый опыт и множество мыслей для улчшения кода.
Слой **Composite Root** реализован не идеально, в особенности из-за **Odin**, но выдерживает практики хорошего кода. Проект показал насколько гибкой можно делать
архитектуру.
